Recent Projects

UNISTAY-NAVIGATOR WEBAPP

Dynamic-responsive Website

I developed Unistay Navigator, a full-stack mobile-first platform built with React.js, TypeScript, and Supabase (PostgreSQL, Auth, Storage). The application enables university students in Uganda to efficiently discover and compare private hostels and residential accommodations. It also integrates dynamic modules for campus news, events, job listings, and automated weekly student spotlight features (MCM/WCW)!

USRA WEBSITE

I built a dynamic full-stack website for the Uganda Schools Rugby Association (USRA) using React.js, JavaScript, and Supabase (PostgreSQL, Auth, Storage). The platform provides comprehensive information about the association, features a school and player registration system, and includes a repository of important downloadable documents along with other administrative and informational tools!

KAMPANIS-SBC WEBSITE

Static-responsive website

I developed a responsive static landing-page website for Kampanis Shoes & Bags Clinic. The site features a pinned shop location, a lazy-slide top picks gallery, and multiple structured pages showcasing services and products. Built with React components and lightweight HTML/CSS/JS for fast performance and clean presentation!

EXCEL CONSTRUCTION
